# Sales Inventory: X-Bakery

Simple X-Bakery Sales Inventory Management App.

Features:
1. Authentication
    - Sign In
    - Sign Up
    - Reset Password using OTP
    - Profile Manage
2. Customer Management
3. Category Management
4. Product Management
5. Invocie Management
6. Dashboard Summary
7. Report Generation
